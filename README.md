# IT3040 Assignment 1 – Playwright Test Automation

This repository contains automated test execution for Singlish to Sinhala transliteration using Playwright and Python.

---

## 📁 Project Files

- IT23363502.py – Playwright automation script
- IT23363502.xlsx – Excel file containing test cases and results

---

## 🧰 Prerequisites

Make sure the following are installed:

- Python 3.11 or later
- Google Chrome (recommended)

---

## ⚙️ Installation Steps

1. Clone the repository:

git clone https://github.com/AsiriTenne/IT3040_Assignment_1_IT23363502.git  
cd IT3040_Assignment_1_IT23363502

2. Install required dependencies:

pip install -U pip  
pip install playwright openpyxl  

3. Install Playwright browsers:

playwright install  

---

## ▶️ Running the Test Automation

Run the following command:

python IT23363502.py  

---

## 📊 Results

- The script updates the Excel file automatically with:
  - Actual Output
  - Status (Pass/Fail)

---

## 📝 Notes

- Ensure the Excel file is in the same directory as the script
- Close the Excel file before running the script
- Keep the browser open to observe automation
